
# Easy Log


Management software for small businesses (Alpha 0.1v)



## How to use:

#### At firts, you have to clone the project:

```http
  git clone https://github.com/joaodiogogithub/Easy_LOG.git
```


#### After that, now just run:
```http
  python manage.py runserver
```
